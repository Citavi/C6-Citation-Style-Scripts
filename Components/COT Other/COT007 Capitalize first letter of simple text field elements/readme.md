# Capitalize first letter of simple text field elements
Alle Wörter im Titelfeld sollen mit einem Großbuchstaben beginnen.

## Prerequisites
Citavi 6 (and above)

## Use Case Example 

- Doe, J. (2012): Living with Dogs Today. Köln.
