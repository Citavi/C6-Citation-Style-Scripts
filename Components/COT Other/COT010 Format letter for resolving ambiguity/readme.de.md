# Format letter for resolving ambiguity
Mehrdeutigkeitsbuchstaben frei plazieren und formatieren.

## Voraussetzungen
Citavi 5 (oder höher)

## Beispiele
Citavi hängt bei Mehrdeutigkeiten, also für den Fall, dass ein Autor mit mehreren Publikationen aus dem selben Jahr im Text zitiert wird, einen Buchstaben an das Jahr (Miller 2010a, 2010b); einzustellen über das Menü **Eigenschaften des Zitationsstils** > **Mehrdeutige Nachweise**. Die Formatierung des Buchstabens ist nicht anpassbar.
Dieses Skript ist für den seltenen Fall gedacht, dass der Buchstabe besondere Formatierungen benötigt (Miller 2010 a) bzw. frei im Text platziert werden muss.

## Anpassung
In den Eigenschaften des Zitationsstils ist einzustellen, dass der Buchstabe für die Mehrdeutigkeitsunterscheidung an das Feld **Kurzbeleg** angehängt werden soll (auch wenn das im Stil nicht verwendet wird). Hinter das Feld **Jahr ermittelt** im Literaturverzeichnis sowie im Kurznachweis im Text ist ein zusätzliches Textelement anzuhängen, das keinen Inhalt hat. An dieses Textelement ist der hier bereitgestellte Programmcode zu hängen.
